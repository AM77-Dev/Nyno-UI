# Colors

The available color classes are:
    - dark
    - light
    - green
    - blue
    - purple
    - red
    - orange
    - yellow
    - brown
    - teal
    - pink


The available hovers classes are:
    - _dark      /* the dark color of the used color  */
    - _light     /* the light color of the used color  */
